# Portfolio

Simple, research-focused static portfolio site.

## Edit quickly

- Main content: `index.html`
- Styling: `styles.css`
- Script: `script.js`
- Personalization checklist: `changes.md`

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.
